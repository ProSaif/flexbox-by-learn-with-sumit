Flex Container Properties

flex-direction

  1. raw (by default)

  2. colum

  3. raw-reverse

  4. colum-reverse

flex-wrap

  1. nowrap (by default)

  2. wrap

  3. wrap-reverse

flex-direction + flex-wrap = flex-flow

css flex code ca be written by this way:

flex-flow: flex-direction flex-wrap

Example:

flex-flow: colum-reverse wrap
